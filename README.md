# RT-MAP

Real time map manager


## Prerequisites

* Python 2.7 branch ( sudo apt-get install python )
* Python-gdal 1.11 or greater ( sudo apt-get install python-gdal )
* Git 2.7.4 or greater (sudo apt-get install git)
